Content

Overview of the book

  1. Encode information by binary code and relative operation
  2. Physical layer (electron to transistor) and combine them to form the more complicated device -> combine these structure to von Neanamm machine (in this book: LC-3 aka. little computer 3)
  3. program in LC-3 using our own language
  4. program in LC-3 using assembly language
  5. program in LC-3 using data structure
  6. deal with the I/O in LC-3
  7. C/C++ language

Two core notion in this book

  • The notion of abstraction
  • Don't treat hardware and software sperately

Two very important ideas

  • All computer can do the same things if enough memory is given
  • Problem should be transformed for computers to solve
